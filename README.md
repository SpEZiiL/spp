# Script Preprocessor #

![status][status-img]
![version][version-img]
[![lang: C][lang-c-img]][lang-c-link]

A preprocessor for scripts written in **C**.

**spp** is mainly supposed to be used to build big scripts that have been split
up into several pieces.
